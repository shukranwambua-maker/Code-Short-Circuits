This is a fun hobby project to improve the "GENERIC" and mostly available RC Cars found in the market. If this works, I'd satisfy a childhood dream as well as enable maybe hundreds of other little boys enjoy RC toy casrs. It is made up of an ESP32, L298N Motor Driver, SG90 servo motor & A 7.4v 18Wh Li-ion Battery pack. Hopefully I'll improve this iteration in the future and try to make it economically viable for little boys.
